Como funciona

Tres pasos. Sin timeline. Sin interfaz.

01

Describe

Dile a Claude que video quieres. "Hazme un TikTok anunciando mi nuevo curso" o "agrega subtitulos a mi video."

02

Claude escribe

Renderiza

How it works

Three steps. No timeline. No GUI.

01

Describe

Tell Claude what video you want. "Make me a TikTok announcing my new course" or "add captions to my talking head video."

02

Claude writes

Claude creates a Remotion composition using 25 pre-built components — text animations, backgrounds, overlays, transitions, and more.

03

Render

Preview in Studio, tweak with feedback, then export as MP4 for any platform — TikTok, Instagram, YouTube, Twitter.

Capabilities

Everything you need to edit.

25 Components

Full component library

Animated titles, lower thirds, gradients, particles, split screens, picture-in-picture, transitions, and more. All composable.

AI Captions

Whisper transcription

Local AI generates word-level captions from your video. TikTok-style highlighting with 5 visual presets. No API keys.

Smart Cuts

Silence removal

FFmpeg detects dead air in your footage. One component removes it all — keeping only the speech segments with smooth transitions.

Multi-Platform

Render for any format

TikTok (1080x1920), YouTube (1920x1080), Instagram, Twitter — batch render one video for every platform in a single command.

Visual References

Style browsing

The agent browses the web for visual references, styles, and trends. Say "I want the Apple style" and it searches for you.
